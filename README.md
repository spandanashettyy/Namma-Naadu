# Namma Naadu – Karnataka Travel Exploration Platform

A full-featured travel web application for exploring Karnataka's destinations, itineraries, festivals, and travel tips.

## Features
- **12 destination cards** with categories: Heritage, Nature, Wildlife, Beach, Hills, Spiritual
- **Live search & filter** by name, description, tags, and category
- **Interactive destination modal** with highlights and travel tips
- **5-day Karnataka itinerary** with timeline layout
- **8 festival cards** with dates and descriptions
- **6 travel tip cards** covering best time, food, transport, stay
- **Enquiry/contact form** with validation
- Fully responsive design (mobile + desktop)
- Pure HTML, CSS, JavaScript — no frameworks required

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- GitHub version control (ready to push)
- No backend needed — open directly in browser

## How to Run
```bash
# Simply open in any browser
open index.html
# OR
python -m http.server 8080   # then visit http://localhost:8080
```
